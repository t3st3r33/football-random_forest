# football-random_forest

Proyek ini bertujuan untuk mengembangkan model Machine Learning yang mampu memprediksi hasil akhir pertandingan sepak bola internasional (Menang, Seri, atau Kalah). Dengan memanfaatkan data historis pertandingan dari tahun 2010 hingga sekarang, proyek ini menerapkan teknik Data Mining menggunakan algoritma Random Forest Classifier untuk mengidentifikasi pola kemenangan berdasarkan identitas tim dan jenis turnamen.
